dotfiles
========
dotfilesの整理

下記をターミナルで実行
-------
* chmod +x ./symlink.sh
* bash ./symlink.sh

### fish
* brew install fish
* sudo sh -c "echo /usr/local/bin/fish >> /etc/shells"
* chsh -s /usr/local/bin/fish
* fish
* curl https://git.io/fisher --create-dirs -sLo ~/.config/fish/functions/fisher.fish
  *(https://github.com/jorgebucaran/fisher)*
* fisher
